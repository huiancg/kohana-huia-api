#### Operações
Por padrão é possível fazer quase todas as operações de teste no banco:
~~~
  'queries' => array(
    
    'operations' => array(
      '=', 
      '<', 
      '<=', 
      '>', 
      '>=', 
      '!=', 
      'in', 
      'between', 
      'not in', 
      'not between', 
      'like', 
      'null', 
      'not null'
    ),

  ),
~~~
